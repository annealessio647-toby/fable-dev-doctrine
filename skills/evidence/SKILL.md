# 证据收敛环（Evidence Loop）

**分工声明**：本操典提供控制流、判据与出口路由；每一步的领域手法（具体查什么表、
做什么实验）由执行者用自己的领域知识实例化——操典保证你不绕圈、不冤案、不烂尾。

一句话：**叙事是嫌疑人，证据是法官；每一轮行动的目的是让假设空间单调收缩。**

## 循环本体

```
现象锚定 → 主张翻译 → 全空间列举 → 最便宜的判别器 → 执行 → 剪枝留档 → 收窄
    ↑____________________________ 重复，空间不许无新证据地变大 ____________|
    收窄到单点后 → 走【终局三出口】之一收案
```

1. **现象锚定**：先把现象本身钉死成数字与关系（谁、何时、多少、和什么同时变）。
   若现象是复合指标，先做因子分解（总量=数量×单价这类恒等式），矛盾往往在
   分解后自动收窄到某一个因子。
2. **主张翻译协议**——任何"某人说/报告称/文档写"的说法一律走三步转成假设：
   ① 追问补全到**带时点、带数值、带来源**的具体断言（"口径变了"→"X 日上线的
   Y 促销把折扣从 a 改成 b"）；② 映射到嫌疑清单的某一层（映射不上=清单漏了层，
   补层）；③ 为它写专属指纹和可证伪预言。走完三步它才是待验假设；走不完的说法
   只配当线索，不配当证据。
3. **全空间列举**：把现象经过的每一层/每个环节列成嫌疑清单——注意列的是**现象
   经过的全空间**，不是"有人主张过的空间"（没人提的层往往就是真凶藏身处）。
   给每个嫌疑写出它"作案"时的专属指纹（会留下什么可观测量、什么时间相关性、
   什么量纲特征）。写不出指纹的嫌疑=你还不理解它，先补理解。
4. **判别器**：一个好实验让两个候选给出**相反**预期，一次剪一半；两个候选预期
   相同的实验是零信息，不许跑。按成本从低到高选：**利用现成的对照物**（另一条
   独立记录通道、另一台同类设备、旧版本重跑今天的输入）→ **只改输入再观察**
   （换命令、换批次、换时间窗）→ **增加观测点**（在边界上加记录）→ **改造系统
   本身**（最后手段）。设计完自问："这个读数出来，哪两个解释会分道扬镳？"
5. **剪枝留档**：排除（洗白）记录与定罪同等重要——写下"层 X 无罪，证据 Y"，
   同一嫌疑不许无新证据地二次受审。这让搜索单调收敛而不是绕圈。

## 终局三出口（收窄到底后按情形选一，选不进=还没到底）

- **出口 A · 单一根因定罪**：证据落到最底层（字节、原始记录单行、现场状态值），
  "高度吻合的解释"不算。验收=修复后原复现实验从必现变零现，且全量回归无损。
- **出口 B · 多因并存分摊**：几方主张各对一部分时，用守恒式把总效应定量分解到
  各已证实因子（因子 1 占 x%、因子 2 占 y%），**残差小于观测噪声才许收案**——
  噪声基准取该指标在历史无事件期的正常波动幅度；残差大于它说明还有未列举
  的层，回到循环。
- **出口 C · 非缺陷结论**（真因是业务事实/环境变化，无所谓"修复"）：验收=用该
  结论**正向重构**出观测到的全部现象含量级，且至少一条独立通道交叉证实。
  重构不出量级=解释不完整，不许收案。

## 注意力法则（高频指纹速查，均为原理"矛盾即模型误差信号"的实例）

- **反直觉数据是门，不是噪声**："负载越重反而越快"指认空闲路径（轻载命中了
  空闲等待/休眠的慢分支）；"误差恒等于工作点的固定百分比"指认统计口径而非
  物理噪声（真噪声不会恰好正比于设定值）。抓矛盾，别抓佐证。
- **"重试就好了"是消费端缺陷的经典指纹**——东西一直在，是取的人不认识它。
  凡"第二次总能成功"，先审读取/解析的一方。
- **"监控显示正常"先审监控**：计数器为零要先证明计数器在当前负载下可信
  （队列溢出、日志截断、聚合显示只露头部——观测通道自己也是嫌疑层）。
- **偶发 ≠ 随机**：找齐全部发作实例的公共因子（时间点、前置动作、状态），
  公共因子=触发条件=复现器。连续型问题看时间形态：阶跃指向"那一天上线的
  东西"，渐变指向趋势性原因。

## 真相的两级制

事实只分两级：**亲自查证过唯一源的**，和**其余一切（记忆、转述、"应该是"）**。
只有第一级能当推理地基。

- 凡有唯一源的事实（代码、文档、测量、原始记录）一律现查，禁止凭记忆/惯性
  引用；记忆只用来定位唯一源在哪。
- **测量先于争论**：理论分支超过两个还在增殖时，停止推理去造一个测量。
  一次测量的成本 < 三个错误理论的成本。

## 对抗自己的叙事

- 每个漂亮解释诞生时立刻写出**可证伪预言**，下一个实验优先尝试杀死它。
  解释活过刺杀才配升级为结论。
- **被质疑时先找自己推理的洞，再辩护**。对方一句"只有一台设备在用，哪来的
  拥挤？"可能正好戳穿你从未测量过的默认假设（例中：把延迟归咎于信道拥塞，
  却从没测过信道占用）。
- **环境优先**：昨天好、今天坏、期间零改动 ⇒ 先问"物理/环境/别人动过什么"
  再查自己的东西。持有系统外状态的人是最便宜的传感器——早问、按主张翻译
  协议的粒度具体地问。

## 交付物

结案产出**判决书**：出口类型（A/B/C）+ 根因或分摊表 + 每层排除证据 + 验证数字 +
**哨兵指标**（此病复发立刻报警的日常观测量）+ 本案新增铁律。写给三个月后失忆的自己。
